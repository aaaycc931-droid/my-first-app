# AGENTS.md

## 项目目标
这是一个Next.js应用，用于逐步构建MVP产品。

## 开发原则
- 只做MVP，不做复杂优化
- 每次只实现一个功能
- 优先保证可运行，而不是完美
- 所有改动必须可解释

## UI原则
- 简洁优先
- 不要过度设计
- 先功能后美观

## 技术栈
- Next.js
- TypeScript
- TailwindCSS

## 工作方式
- 逐步开发
- 小步提交
- 避免大规模重构
- 开始改代码前，必须先阅读与当前需求直接相关的代码、文档和配置；如果需求涉及多个模块、项目方向判断或不确定的仓库结构，先做只读扫描或让 Codex 做 read-only audit，再决定改动方案
- 不得在没有完整扫描仓库或相关上下文时声称“已经了解全部代码”；如未扫全，必须明确说明已阅读范围、未知范围和下一步需要确认的内容
