# Splendid Grand Piano mobile sample set

- Source instrument: Splendid Grand Piano (Steinway), Public Domain.
- Browser-ready OGG conversion: `danigb/samples`, commit `eae8c5c7a10f4b0e06281211e704c36c30e25342`.
- Source URL: <https://github.com/danigb/samples/tree/eae8c5c7a10f4b0e06281211e704c36c30e25342/audio/splendid-grand-piano>
- Packaging: 12 root notes across A0–A#6 in each of PP, Mp and Mf velocity layers. Runtime zone mapping covers the full A0–C8 MIDI range.
- The compatibility synthesizer remains available as an offline fallback if an asset cannot be decoded.

The upstream `samples.json` declares the instrument and its samples Public Domain.
