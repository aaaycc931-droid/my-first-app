# P119 审核计划与 finding 修改记录（私有填写模板）

> 填写版不得提交 Git、上传聊天、进入 CI artifact 或打包进 APK。仓库只保留空模板。

| 版本 | 时间 | 触发 finding / 原因 | 受影响 kind × difficulty / item | 修改内容 | 是否重建 manifest | 是否重建候选 | 两位教师重新预批准引用 | 状态 |
|---|---|---|---|---|---|---|---|---|
|  |  |  |  |  |  |  |  |  |

状态只允许：

- `OPEN`
- `FIXED_PENDING_REVIEW`
- `REAPPROVED_PENDING_REVIEW`
- `CLOSED`

禁止静默替换已批准 item ID、manifest、seed 或分层。任何影响题目、答案、解释、难度或课程内容的修复都必须生成新的 commit-qualified manifest，再建立新候选版本并由两位教师重新预批准。
