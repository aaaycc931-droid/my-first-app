## 当前边界

本目录是 docs/assets-only 冻结，不表示伙伴运行时已经完成。当前不得据此宣称已实现：

- 场景内悬浮伙伴 UI；
- 表情、动画或状态机；
- 语音合成、语音识别或生成式对话；
- 长期记忆、账号同步或云端人格；
- 自动读取全部练习数据；
- 角色剧情系统。

未来实现必须复用项目现有 `CompanionContext` / 活动证据安全边界，不得虚构练习分析，也不得把低置信证据包装成确定判断。

## 禁止回退

探索阶段的全身形态、机甲碎甲、黑色大头、眼状侧孔、长脖子、毛发、披风、翅膀和多余尖刺均已废弃。第三方角色图片只曾用于讨论抽象设计语言，不得复制、打包或进入产品资产。
