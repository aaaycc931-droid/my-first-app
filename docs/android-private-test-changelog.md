# Android 本地私测版本变更记录

本文件只记录可追溯的 Android 本地私测候选。调试签名包用于产品所有者私下安装测试，不是应用商店包、专用 release 签名包或正式评分产品。

## 0.2.0（versionCode 2）

构建状态：P95 候选已通过 CI；P96 三难度本地题库切片待 CI。

### 用户可见变化

- P89：四类独立本地练习改为会话内随机题序。同一会话内每道题可复现，并会先走完整题库再循环；关闭应用后重新生成题序，不形成学习历史或难度自适应。
- P90：新增 Android 本机错题复练。查看答案后，答错题加入并置顶，答对题移除，最多保留 12 项；首页可进入复练，并可通过页面内二次确认清空。本机只保存复现题目所需的最小信息，不保存作答、答案、音频、分数、准确率、账号或完整历史。
- P91：下载候选包前增加独立工件复核门禁。CI 会重新计算 APK 摘要与字节数，并交叉核对校验文件、构建报告、版本和 commit；缺失、多余或不一致会阻止上传。这是交付可靠性增强，不改变练习评分边界。
- P93：新增真实挂载移动 React App 的本机复练行为回归，自动覆盖错题加入/移除、清除确认、历史导航和存储失败降级。
- P94：新增离线“本地参考钢琴”独立页，并在四类练习中提供默认收起的找音入口。包含三档 13 键音域、多指、音量、延音、全停和中文失败重试；弹奏不保存、不上传，也不生成成绩。
- P95：本机复练新增稳定题目标识，已有 v1 记录会在本机原地升级为 v2，避免后续题库扩容时旧错题指向另一题；迁移失败提示不阻断练习。本切片未扩充三难度题量，也未改变课程或云端 v1。
- P96：Android 四类听辨练习均增加基础、进阶、挑战三档，每一题型每档至少 20 个版本化组合；手动难度通过音域、半音、跨度、节奏型、速度和旋律跳进区分。共享 Web、课程和云端 v1 不变；该难度仍是开发者编排练习，不冒充教育校准或正式自适应评估。

### 已知限制与安装提醒

- 当前候选仍使用 Android Debug 调试签名，不是专用 release 签名正式包。
- 尚未完成真实 Android 手机的安装、断网冷启动、扬声器音频、跨重启复练、后台恢复和长循环验收；自动化 PASS 不等于真机 PASS。
- 安装 0.2.0 前需先卸载旧 debug 包，再安装新 APK。卸载会清除旧包的本机复练队列；请勿把它当作覆盖升级或数据迁移已验证的证据。
- APK 不申请网络、麦克风或存储权限；不依赖 Vercel、Supabase 或远程 `server.url`，也不提供账号、云同步或正式评分。
