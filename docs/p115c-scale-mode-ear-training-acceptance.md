# P115c：音阶与调式听辨验收标准

状态：**IMPLEMENTATION CANDIDATE / 本地严格验证中**
日期：2026-07-19
QA level recommendation：**strict**

## 1. 目标

在 Android 离线练耳入口交付 P115 的第三个纵向切片：用户主动播放从主音上行到高八度的音阶，辨认音阶或调式类型，再进入统一 Activity 非评分答案、本机复练和学习事实画像闭环。

## 2. 题库与难度

- 基础：自然大调、自然小调、大调五声、小调五声，覆盖 12 个主音，共 48 个稳定变体。
- 进阶：基础四类加和声小调、旋律小调上行形态、多利亚、混合利底亚，覆盖 12 个主音，共 96 个稳定变体。
- 挑战：进阶八类加弗里几亚、利底亚、洛克里亚与全音音阶，覆盖 12 个主音，共 144 个稳定变体。
- 伊奥尼亚由自然大调承载，爱奥尼亚由自然小调承载；由此覆盖七种教会调式而不制造同音异名的重复答案。
- 每个变体由确定性半音结构生成；同一会话完整遍历前不得重复，相同种子、题号与稳定目标必须复现相同题目。

三档题量都超过长期路线每档至少 40 个稳定题目或等价组合的数量门槛，但在两名视唱练耳教师审核难度、术语与解释前，不得宣称教育审核或完整 P115 完成。

## 3. 真实使用路径

1. 用户选择基础、进阶或挑战，主动播放上行音阶。
2. 各音按固定间隔依次播放；停止、新题、切换难度、离开页面和后台生命周期必须停止既有音源。
3. 用户选择音阶或调式；查看答案后锁定选择，显示主音、音程结构、特征音和非评分解释。
4. 错题以 `scale-mode` 稳定目标写入本机复练；复练答对后移除。
5. 查看答案与开始复练写入最小学习事件，只增加该题型事实，不推断能力等级。

## 4. 迁移与隐私

- 复练队列升级到 schema/catalog v5，迁移 v1–v4；旧版本不得伪造音阶/调式目标。
- 学习画像信封升级到 schema v4，迁移 v1–v3 并为 `scale-mode` 事实补零。
- 不保存音频、录音、用户具体选择、账号、设备标识、分数或能力标签；不上传、不联网、不写数据库。
- 损坏、超限、未知字段、未知版本或与难度不匹配的稳定目标必须失败关闭。

## 6. 不包含

- 降阶音阶、爵士旋律小调派生调式、微分音、自由定制音阶或下行旋律小调形态；
- 麦克风/MIDI 作答、正式评分、自适应、跨设备同步或伙伴运行时；
- 两名教师审核、五名目标用户、三档 Android 真机矩阵、正式签名或最终发布完成。
