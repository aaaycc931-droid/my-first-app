# P55 — 内置音程听辨 Runtime Alpha 验收标准

## 范围

在 `/practice` 增加一个独立“听辨练习”功能区，提供浏览器本地、会话内的内置音程听辨题。用户可播放两个依次出现的合成音、选择音程、查看本题答案说明、重置本题并进入下一题。

## 必须满足

- 默认 `/practice` 入口仍是“本地旋律”。
- 用户可在“听辨练习”进入基础或进阶难度。
- 基础题只包含大三度、纯四度、纯五度；进阶题可包含二度、三度、纯四度、纯五度。
- 题目音频由当前浏览器 Web Audio 合成；播放、停止、重播可用，并对无法播放给出简体中文提示。
- 未选择答案时不能查看本题答案，并有中文原因。
- 查看答案后同时显示标准答案、简短解释和用户选择；不显示分数、百分比、等级、通过或失败。
- 重置本题清除选择和答案说明；下一题切换到下一道内置题并清除旧状态。
- 刷新后所有题目状态消失；不写入 localStorage、IndexedDB、账号、数据库或云端。

## 非目标

不实现上传、录音、麦克风、真实演唱识别、正式听辨评分、错题持久化、账号、云端、数据库、题库下载、新依赖、`/api/recognize`、OCR / OMR 或最终目标。

## QA

建议 QA level：strict。应运行 focused helper test、build、`git diff --check`，并在合并后的生产环境实际验证播放、选择、查看答案、重置、下一题、刷新和无持久化 / 无网络请求。
