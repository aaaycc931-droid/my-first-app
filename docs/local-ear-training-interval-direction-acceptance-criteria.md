# P56 — 本地音程听辨方向验收标准

P56 扩展 P55 的内置音程听辨：用户可在当前页面会话中选择“上行”或“下行”题目。上行按低音到高音播放；下行按高音到低音播放。题目仍来自确定性内置题库，答案只说明本题音程和用户选择是否一致。

## 用户流程

1. 用户进入“听辨练习”，默认使用基础、上行题目。
2. 用户可主动切换上行或下行；切换会重置当前题的选择、答案可见状态和题目序号。
3. 点击播放后，浏览器本地 Web Audio 按所选方向依次播放两个正弦音。
4. 用户选择音程、查看本题答案、重置或进入下一题。

## 边界

- 只支持内置音程的上行 / 下行方向；不实现旋律听写、题库下载、录音、麦克风或用户音频分析。
- 不产生分数、准确率、等级、通过 / 失败、正式测评或练习历史。
- 题目、方向、选择和答案说明仅保留在当前页面内存；刷新后消失。
- 不上传、不调用 API 或 `/api/recognize`，不写入 localStorage、IndexedDB、账号、数据库或云端。

## QA

- focused tests 覆盖上行和下行频率映射、题目 id 区分、答案说明与非评分字段边界。
- 浏览器 QA 覆盖方向切换重置、下行播放状态、答案、重置、下一题和刷新后的默认入口。

QA level：strict。
