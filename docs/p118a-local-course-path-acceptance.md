# P118a — 中文本地课程路径与课程进度验收

日期：2026-07-22

状态：implementation candidate

QA level recommendation：**strict**

## 1. 目标

P118a 在 Android 本地模式中建立第一条可运行的中文课程路径。课程目录、章节、课节、学习目标与 Activity 映射全部随安装包版本化；用户不登录、不联网也能查看课程、开始当前已解锁课节，并在完成课节规定的“核对一次活动结果”后记录本机完成事实和解锁下一课。

该进度只说明哪些课节已按当前课程规则完成，不是能力评分、等级、通过／失败、天赋判断、教师结论或教育有效性证明。

## 2. 课程目录与稳定身份

- 课程、章节、课节、目标和 Activity 映射必须使用稳定、版本化且互不重复的 ID；显示标题不得参与身份判断。
- 首版至少包含一门简体中文入门课程、两个有序章节和多个有序课节，并只映射到已经存在且能产生已核对结果的本地 Activity。
- 所有课节引用必须指向同一目录中的章节、目标和允许的 Activity；缺失、重复、乱序、跨课程引用或未知 Activity 必须失败关闭。
- 内容版本升级时，只有显式登记且仍能映射到当前稳定课节 ID 的旧进度可以迁移；不能证明兼容时必须拒绝旧记录并从空进度继续。

## 3. 完成条件与解锁

- 首版完成条件固定为：用户从当前课节入口开始其映射的本地 Activity，并主动查看一次结果；答案正确与否都只构成“已完成本课规定活动”的事实。
- 不得把 `isCorrect`、错误次数、复练队列、音高偏差或学习画像计数转换为课程分数、能力等级或通过／失败。
- 第一课默认解锁；后续课节按目录顺序要求前一课已完成。锁定课节必须显示解锁条件且不能被 UI 或伪造本地进度绕过。
- 重复完成同一课节必须幂等；陈旧目录版本、错误 Activity、锁定课节、未知课节和迟到结果不能推进进度。
- 切换页面、后台、重置、卸载或陈旧回调不得把一个 Activity 的结果归到另一课节。

## 4. 本地进度与隐私

- 进度 envelope 必须版本化，并绑定课程 ID、课程内容版本和已完成课节 ID；只保存完成事实和必要修订号。
- 不保存答案内容、正确／错误、分数、ActivitySession、复练目标、录音、Blob、PCM、音高帧、分析曲线、设备标识、账号或其他身份数据。
- 存储不可用、读取失败、保存失败、超限、损坏、未知字段、旧版本不兼容或内容版本不匹配时必须安全降级；核心随机练习继续可用。
- 用户可以单独重置课程进度；不得连带清除复练队列、学习画像、录音或其他本机数据。
- 不新增账号、云端、数据库、网络依赖、上传、遥测或 Android `INTERNET` 权限。

## 5. Android UI 与生命周期

- Android 首页提供清楚的“中文课程”入口，与随机练习、定制练习、复练和学习画像区分。
- 课程页显示章节、课节目标、完成条件、已完成／当前可开始／锁定状态和本机隐私说明。
- 开始课节时只进入该课节声明的既有 Activity；离开、返回课程、后台恢复或开始其他模式必须清除旧课节归属。
- 当前课节结果被核对后，进度保存成功才显示已完成并解锁下一课；保存失败时不得仅在 UI 中假装已经持久完成。
- 重置课程进度必须二次确认，并在失败时保留原记录和给出非阻塞说明。

## 6. 自动门禁

- focused domain：稳定 identity、目录结构、引用完整性、顺序解锁、完成幂等、错误 Activity／锁定／陈旧版本拒绝、旧版本安全迁移与序列化上限。
- storage：空记录、保存／读取、迁移写回、损坏清除、读写删异常、无存储降级、保存失败不破坏旧记录。
- mounted mobile：首页入口、课程章节与状态、锁定按钮、开始当前课、结果核对后完成与解锁、返回／后台／重置清除旧归属、存储失败不冒充完成。
- 回归：既有随机练习、定制、复练队列、学习画像、P116/P117 Activity、全局音频和 Android 生命周期行为保持不变。
- `npm audit --omit=dev --audit-level=high`、`npm run check`、Android sync/validator、GitHub `quality` 与 `android-local` 必须通过。

## 7. 外部证据

当前 Android 三档真机、教师课程结构审核和至少 5 名中文目标用户的课程可用性证据均为 `NOT_EXECUTED / INCOMPLETE`。自动测试、DOM、CI 和 Debug APK 不能替代这些证据。

P118a 自动门禁通过只代表中文本地课程路径 implementation candidate；不代表课程内容充分、教学有效、能力测量有效或正式 V1 完成。
