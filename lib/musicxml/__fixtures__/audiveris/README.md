# 真实 Audiveris fixtures

这里未来存放由 Audiveris 实际导出的 MusicXML 样本。

- 当前不要在此目录放置 synthetic fixture。
- 每个真实样本都应配套同 basename 的 `.expected.json` 和 `.source.md`。
- `.source.md` 必须记录样本来源、脱敏情况、原始图片保留情况、已知的 Audiveris
  版本与导出时间，以及是否允许提交到仓库。
- 来源不能确认时，不得将样本标记为 `audiveris-real`。

详细接入流程见 `docs/audiveris-sample-intake.md`。
